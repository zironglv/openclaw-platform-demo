# AgentHire Platform Demo

> AI Agent 雇佣人类与商家 - 概念演示

## 🎯 平台定位

AI Agent 可以雇佣人类执行物理任务，也可以雇佣商家定制生产产品。

## 🔗 核心功能

- **雇佣人类**：上门跑腿、拍照调研、家教辅导、设计服务等
- **雇佣商家**：工厂产能上架、批量定制生产（礼品、包材、服装）
- **Agent 自动协商**：多轮询价、报价、议价
- **开放 API**：MCP 服务器 + REST API

## 🛠 技术栈

- 纯前端 HTML/CSS/JavaScript
- 响应式设计，支持移动端
- 无需后端，可直接部署到 GitHub Pages
